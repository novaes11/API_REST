# 📚 Repositório de Estudos - API REST em Node.js (Faculdade)

Bem-vindo ao meu repositório de estudos! 🎓

Este espaço documenta meu aprendizado prático na construção de **APIs RESTful** com **Node.js** durante a faculdade.

## 🎯 O que foi aprendido
- Estruturação de APIs baseadas na arquitetura REST.
- Implementação de operações CRUD (Create, Read, Update, Delete).
- Uso correto de verbos HTTP e Status Codes.
- Validação defensiva de dados e tratamento de erros.
- Testes de requisições e automação com Postman.
- Integração com banco de dados relacional (SQLite).
